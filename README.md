# Telegram-бот для записи к мастеру маникюра (aiogram + SQLite)

## Возможности
- Запись клиента через inline-кнопки.
- Календарь доступных дат на 1 месяц вперед.
- Выбор только свободного слота.
- Один активный слот на пользователя.
- Сохранение записи в SQLite.
- Уведомление администратору и публикация в канал.
- Отмена записи пользователем (слот снова становится доступным).
- Проверка подписки на канал перед записью.
- Кнопки `Прайсы` и `Портфолио` в главном меню.
- Админ-панель:
  - Добавление рабочих дней.
  - Добавление/удаление слотов.
  - Закрытие/открытие дня.
  - Просмотр расписания на дату.
  - Отмена записи клиента.
- FSM для всех диалоговых сценариев.
- Автонапоминания за 24 часа через APScheduler.
- Восстановление задач напоминаний после перезапуска.

## Настройка
### 1) Создать Telegram-бота через @BotFather
1. Откройте Telegram и найдите `@BotFather`.
2. Отправьте команду `/newbot`.
3. Придумайте:
   - **имя бота** (любое, например `Nail Master Bot`)
   - **username** бота (должен оканчиваться на `bot`, например `my_nail_master_bot`)
4. BotFather пришлёт **токен** вида:
   `1234567890:AA...`
   Это и есть значение для `BOT_TOKEN`.

### 2) Узнать свой Telegram ID (для ADMIN_ID)
1. Найдите в Telegram бота `@userinfobot` (или `@getmyid_bot`).
2. Нажмите Start.
3. Скопируйте ваш числовой ID (например `123456789`).
4. Это значение для `ADMIN_ID`.

### 3) Создать канал и получить CHANNEL_LINK / CHANNEL_ID
Если вы никогда не делали канал — вот самый простой путь:

1. В Telegram нажмите **Меню → Создать канал**.
2. Укажите название и описание (можно позже поменять).
3. Выберите тип:
   - **Публичный** — можно задать красивую ссылку `https://t.me/имя_канала`
   - **Приватный** — ссылка будет формата приглашения.
4. Добавьте в канал вашего бота как администратора:
   - Откройте канал → **Управление каналом** → **Администраторы** → **Добавить администратора**.
   - Найдите вашего бота по username и добавьте.
5. `CHANNEL_LINK`:
   - Для публичного канала: `https://t.me/your_channel_name`
6. `CHANNEL_ID`:
   - ВАЖНО: это число вида `-100...`, а не ссылка.
   - Самый простой способ получить:
     1) временно запустите бота без проверки подписки (или с корректным `CHANNEL_LINK`);
     2) отправьте любое сообщение в канал;
     3) вызовите у бота метод `getUpdates` и возьмите `chat.id` канала.
   - Альтернатива: использовать `@RawDataBot` / `@userinfobot` в канале для получения `chat_id`.

### 4) Что вставлять в переменные окружения
Скопируйте `.env.example` в `.env` и подставьте свои значения:
```bash
export BOT_TOKEN="1234567890:AAEXAMPLE_TOKEN"
export ADMIN_ID="123456789"
export CHANNEL_ID="-1001234567890"
export CHANNEL_LINK="https://t.me/your_channel_name"
export DB_PATH="bot.db"
```

Расшифровка:
- `BOT_TOKEN` — токен от BotFather.
- `ADMIN_ID` — ваш личный Telegram ID.
- `CHANNEL_ID` — ID канала в формате `-100...`.
- `CHANNEL_LINK` — ссылка на канал для кнопки «Подписаться».
- `DB_PATH` — путь к SQLite базе (обычно оставьте `bot.db`).
